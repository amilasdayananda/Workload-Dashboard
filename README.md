This project is a Workload Monitoring Dashboard built on top of **Google Apps Script**, **Google Sheets**, and **Gmail**, with a **React + TailwindCSS** frontend rendered via `HtmlService`.

## Features

- Staff & supervisor workload data entry (User / System logs)
- Daily dashboard with charts (completed vs rejected accounts, system volume)
- Performance ratings and star-based scoring per user
- Staff directory & historical performance view
- Export to CSV and HTML email report generation via Gmail

## Tech Stack

- Google Apps Script (Code.gs)
- React 18 (UMD)
- TailwindCSS (CDN)
- Recharts (for charts)
- Google Sheets (database)
- GmailApp (for email reports)
